# joomla_com_categorytag
这是一个joomla的扩展，可以创建某个栏目 AND 某个标签下的文章列表，是一个joomla文章列表扩展

## 注意
由于 Joomla 3.7 以上默认已经支持本功能，因此本项目已经不再维护。

## 说明  
这是一个可以指定一个栏目和一个标签下 (category and tag) 的文章列表组件。
